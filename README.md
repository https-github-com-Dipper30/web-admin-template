# web-admin-template

React.js + Typescript + i18n + react-redux + antd

# Get Started

## Install NPM

```shell
npm install
```

## Build Env File

Create file '.env.development'.
Paste and modify codes below.

```shell
# config
APTX_BASE_URL=http://localhost:3030
APTX_OUTPUT_DIR=dev
```

## Run Project

```shell
npm run dev
```
